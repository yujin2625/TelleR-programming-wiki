# 에셋 파이프라인

## 에셋 타입 분류

| 종류      | 포맷 예시        | 비고             |
| --------- | ---------------- | ---------------- |
| 모델링    | `.fbx`, `.obj`       | 경량 최적화 필요 |
| 텍스처    | `.png`, `.tga`       | 2048 이하 권장   |
| 오디오    | `.wav`, `.mp3`       | 공간감 강조      |
| UI/이펙트 | `.prefab`, `.shader` | 재사용 고려      |

## 파이프라인 흐름

1. 디자이너 제공 원본 에셋 수령
2. Unity 규격에 맞춰 변환 및 압축
3. Git LFS에 등록 후 업로드
