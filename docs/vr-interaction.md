# VR 상호작용 설계

## 입력 매핑

| 동작        | 입력 버튼            |
| ----------- | -------------------- |
| 잡기        | Grip Button          |
| 버튼 누르기 | Trigger              |
| 이동        | Joystick or Teleport |

## 사용 시스템

- Unity XR Interaction Toolkit
- Interactable/Grab/Socket 구성
- Haptic Feedback 적용

## 테스트 체크리스트

- ☐ 물체 잡기 후 릴리즈 작동
- ☐ 두 손 조작 시 충돌 처리
- ☐ 버튼 누름 시 애니메이션 동작
